Dash Platform Sample Dapp's

This is a collection of simple Dash Platform Dapp's to showcase how they can be implemented from third-party developers. These samples use the "Simple Dapp Signing" algorithm used by Chrome Wallet. This is not an official solution!

Checkout the Dash Sample Dapp GitHub repository to learn how to develop your own Dapp using the same method then these samples.

Also follow Dash SDK releases!

Note

Simple Note Dapp as sample for saving documents on Dash Platform using your Dash Username. The deployed data-contract uses objects for date, title, text and encryption status. To use this dapp you need to "Sign in" and then start the Note-Dapp. Create the note, press "Submit" on the webpage and wait for Chrome Wallet dialog to appear. Then confirm the dialog message.

PayMe

PayMe is meant as simple example to submit a payment requested from a website to Chrome Wallet. It uses only your Dash Username and you simply have to "Confirm" it in the Wallet after submission.
